Welcome to the Modular Arithmetic Workbench!

Developed by Ananya Majumdar, under the supervision of Dr. John Longley.

This educational tool has been developed for students and teachers of introductory number theory to act as a plentiful source of examples with visual representations. Please note that this tool is not intended as a standalone resource, it is simply a teaching aid.

Take your time progressing through each screen and play around with numerous combinations of values for the variables you are presented with by changing the slider positions.

  • This button will show incremental progress through the whole animation with each click. This is to give you the time to see the changes between each step.

  • This button will play the full animation from the current state. This is to show you the whole pattern yielded by the group size and generator element you have selected. Note: this button can be pressed after stepping through a few increments with the button above.

  • This button allows you to clear any animation progress even before it has finished playing. Note that this can also be done by changing the size of the group with the top slider.

Each screen is designed to build directly upon what you have seen in the previous one. The visualisations you will see depict the following concepts:

  • Screen 1: Modular arithmetic under addition, generators, additive orders, multiplicative inverses.
  • Screen 2: The structure of additive orders in a group, $\phi(n)$, multiplicative elements.
  • Screen 3: Modular arithmetic under multiplication, multiplicative orders, RSA-friendly elements.
  • Screen 4: The structure of multiplicative orders in a group, primitive roots, cyclic groups.
