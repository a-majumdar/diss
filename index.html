Welcome to the Visualisation Tool for Number Theory

Please note that this tool is not intended as a standalone resource, it is simply a teaching aid.

This educational tool has been developed for students and teachers of introductory Number Theory to act as a plentiful source of examples with visual representations. The repeated exposure to a variety of similarities and differences between concrete examples aims to facilitate the formation of concepts with visual modes of organisation.

To use this tool, take your time progressing through each screen and play around with numerous combinations of values for the variables you are presented with by changing the slider positions. All elements for you to control are on the left-hand side of your screen, and you will see the effect of your changes on the right. The three buttons underneath the sliders also affect the state of the animation:

  • This button will show incremental progress through the whole animation with each click. This is to give you the time to see the changes between each step.

  • This button will play the full animation from the current state. This is to show you the whole pattern yielded by the group size and generator element you have selected. Note: this button can be pressed after stepping through a few increments with the button above.

  • This button allows you to clear any animation progress even before it has finished playing. Note that this can also be done by changing the size of the group with the top slider.
